Welcome to UTrack!

UTrack allows you to track and analyze personal data quickly and easily.

How to use UTrack

UTrack is very simple to use:

  1. Add new data using the Add some data link above.
  2. Analyze your data in various formats using the Analyze your data link.
  3. Modify/remove existing data using the Modify your data link.

Why UTrack?

UTrack can help answer questions like:

  • Am I spending too little time studying?
  • How stressed am I while programming?
  • How happy am I while playing video games?
